# typing Page Project

## 1차 진행
- HTML, CSS를 이용하여 기본적인 화면 구성

## 2차 진행
- JavaScript를 이용하여 기능 구현
  - 사용자가 입력한 문장을 예문과 비교하여 정확도를 측정
  - 문장입력 완료시 결과를 보여줌
- typing game 추가
  - raindrop game
  - card game

## 3차 진행
- API를 이용하여 문장을 랜덤으로 받아옴
  - `https://korean-advice-open-api.vercel.app/api/advice`
- Server와 연동하여 사용자의 점수를 저장

## 추가 기능
- 난이도
- 게임모드
